# Parentrics Team (Webveda Clone)

A responsive, dynamic web application designed to showcase pediatric and parent coaching services, originally inspired by the WebVeda design but tailored for Dr. Abhishek Tiwari and the Parentrics Team. Built using modern HTML, CSS, and Vanilla JavaScript without any heavy external frameworks.

## 🚀 Features

- **Hero Section**: Animated UI elements with dynamic typed text and a custom "comet" particle effect around CTA buttons and video wrappers.
- **Dynamic Service Carousel**: A horizontally scrollable list of service/course cards that dynamically filter based on categories like "New Parents Guide" and "Consultation Sessions".
- **Bento Grid Testimonials**: A responsive, 4-zone Bento box layout featuring:
  - Synchronized and independent card auto-rotation
  - Smooth cross-fade data transitions
  - A real-time looping counter powered by `IntersectionObserver`
  - Floating avatars with randomized keyframe animations
  - A seamless auto-playing video element for visual proof
- **Responsive Navigation**: Includes a mobile-first hamburger menu and sticky header effects.
- **Responsive Design & Mobile Optimizations**: Tailored layouts for tablets and mobile devices to ensure readability and flawless UI across all screen sizes.

## 🛠️ Technologies Used

- **HTML5**: Semantic tags and modern structuring.
- **CSS3**: CSS Grid, Flexbox, keyframe animations, and custom CSS variables for easy theming.
- **Vanilla JavaScript**: DOM manipulation, event listeners, interval timers, and IntersectionObservers for scroll-based animations.

## 🏁 Getting Started

Since this project uses pure HTML/CSS/JS, no build steps are required. 

1. Clone this repository:
   ```bash
   git clone https://github.com/AnkitkCreation/webveda-clone.git
   ```
2. Navigate into the project directory:
   ```bash
   cd webveda-clone
   ```
3. Open `index.html` in your web browser of choice, or use an extension like **Live Server** in VS Code for hot-reloading.

## 📂 Folder Structure

- `index.html`: The main page structure.
- `style.css`: All styling, layout grids, and animations.
- `script.js`: Data logic, interval management, and interactivity.
- `/assets`: Contains all images, SVGs, and videos used on the page.

## 🤝 Contribution Guidelines

We welcome contributions to make this project even better! If you'd like to contribute, please follow these steps:

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/webveda-clone.git
   ```
3. **Create a new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-awesome-feature
   ```
4. **Make your changes** and test them thoroughly.
5. **Commit your changes** with descriptive commit messages:
   ```bash
   git commit -m "Add some awesome feature"
   ```
6. **Push to your fork**:
   ```bash
   git push origin feature/your-awesome-feature
   ```
7. **Open a Pull Request** on the original repository. Please provide a clear description of the changes you've made and the issue it resolves (if applicable).

### Code Style
- Use consistent indentation (2 spaces for HTML/JS/CSS).
- Keep CSS classes semantic and grouped logically.
- Add comments to JavaScript functions explaining their purpose.
